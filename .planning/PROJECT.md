# Fitnivo AI Project

## Context
Fitnivo is a premium, AI-powered fitness and wellness platform. The goal is to build a high-end landing page that communicates a holistic transformation system including Fitness, Nutrition, and Mindfulness.

## Objectives
- Create a stunning, dark-premium landing page.
*   Implement AI-powered visual storytelling.
*   Add a cinematic video scroll scrubbing effect in the hero section.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Vanilla CSS
- Framer Motion
- GSAP (Proposed for video scrubbing)

## Core Features
1.  Hero Section with AI Brain and Video Scroll.
2.  Interactive Pillar sections (Fitness, Nutrition, Mindfulness).
3.  Mindfulness Pulse animation.
4.  Fitbuu AI Coach Mascot.
5.  App Dashboard Preview.
